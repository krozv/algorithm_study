# 11653. 소인수분해

시간 : 1396ms
```python
N = int(input())
a = 2 #소인수
while N  > 1:
    if N%a == 0: #a로 딱 떨어질때!
        N = N//a
        print(a)
    else:
        a = a+1
```
## 설명
예를 들어 N이 152라면?

N=132는 a=2에 나눠 떨어지므로  '2'출력 ,N=132//2

N=66은 다시 a=2에 나눠 떨어지므로 '2'출력, N=66//2

N=33은 a=2로 나눠 떨어지지 않으므로 a=a+1

N=33은 a=3로 나눠지므로 '3'출력, N=33//3

N=11은 a=3으로 안 나눠지므로 a는 계속 1씩 증가하다가

N=11은 a=11로 나눠 떨어지므로 '11'출력, N=11//11

N은 1이므로 while문이 끝남!
